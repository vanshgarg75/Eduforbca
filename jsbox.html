JS-Dialog Box



    →Dialog boxes are popups that can be used for a variety of use cases.
    →types of dialog boxes:
    1. Alert dialog box
    2. Confirmation Dialog box
    3. Prompt dialog box
    Alert Dialog Box:
    →Syntax:
    alert(“Message”);

    Confirmation Dialog Box:


    →Syntax:
    var result_var = confirm(“Message”);
    →this confirm() shows the message and also displays two buttons – Ok and Cancel.
    →the result_var gets true when the user clicks on Ok.
    → the result_var gets false when the user clicks on Cancel.
    →example:

    Prompt Dialog box:


    →In this dialog box, we can get the user to enter some data.
    →Syntax:
    var variable_name = prompt(“Message”);
    →the value entered by the user gets stored in the variable on the LHS.
