    ->HTML is responsible for
    ->the placement / arrangement of the content on the web page
    ->the formatting of the content on the web page
    ->HTML files are of the extension .html or .htm

HTML Rules:

    ->HTML is case - insensitive. It does not treat the upper-case alphabets and lower-case alphabets
    differently. I can write tags in any case.
    ->ie in HTML "a" and "A" are same.
    eg:
    <body>
    <Body>
    <BODY>
    <BoDy>
    there exists no difference between these.
    ->HTML is a tag / elements based language:
    ->tags are ways to give commands or create or place content in a web page.
    ->all the tags are pre-defined. No user defined tags exist in HTML.
    ->HTML does not show any errors. But it might show the correct output.
