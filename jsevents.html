JS-Events


    →JavaScript interacts with the web page using events.
    →events occur when the user / browser changes the web page is a specific way.
    →for example: when the user clicks on a button or when the user hover on top of an image.
    →we can perform the execution of some function at the occurrence of some event.
    →for example: when the user clicks on a button, we want to run the validate().

    Working on the “onclick” event of a button:


    →when the user clicks on a button, we have to run some function.
    →In this example we will also learn how to add some text to a paragraph in the web page.
    →Syntax to add some text to a paragraph:
    document.getElementById(“id_of_the_p_tag”).innerHTML = “String Message”;
    →Syntax to call a JavaScript function when the user clicks on a button:
    <button onclick=”function_name()”>Text

    Working on the “ondblclick” event of a button:


    →when the user double clicks on a button, we can run some function.
    →Syntax to call a JavaScript function when the user double clicks on a button:
    <button ondblclick=”function_name()”>Text</button>

    Working on the “onmouseover” & “onmouseout” events on an element:


    →Syntax to call a JavaScript function when the user brings the mouse over on an element:
    <element onmouseover=”function_name()”>Text</element>
    →Syntax to call a JavaScript function when the user takes the mouse out from an element:
    <element onmouseout=”function_name()”>Text</element>
