CSS

  • CSS: CSS stands for Cascading Style Sheets.
  • CSS is also known as DHTML (Dynamic HTML).
  • CSS is a technology by using which we can apply styles on an HTML document.
  • CSS is for applying "advanced styling" on an HTML document.
  • CSS can be applied to most of the HTML tags.
  • Different ways to apply CSS on HTML tags:
  • 1. Inline CSS
  • 2. Internal CSS
  • 3. External CSS
  • Syntax for applying inline CSS on any HTML tag:
  • <tag style="css_property : value ; css_property : value ;">
  • CSS has hundreds of predefined CSS properties.
  • The style attribute can be used on any HTML tag.
  • CSS3 is the newer version of CSS.
