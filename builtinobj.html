JS-Objects



    →Object is a real-world entity.
    →like a student, car, pen, phone, etc.
    →An object is made up of two things:
    →properties (variables)
    →functions
    →Three ways to create an object:
    1. By object literal.
    2. By creating instance of Object directly using new keyword.
    3. By using object constructor using new keyword.

    Creating Objects using object literal:


    Syntax to create object:
    object_name = { property1 : value1 , property2 : value2 , property3 : value3 };

    Note: we cannot access or display the full object. We can only access or display an object, property
    by property.
    Syntax to access object, property by property:
    object_name.property

    Note: we can expand the object i.e. add new properties to an object using the following syntax:
    object_name.new_property = value

    Creating instance of Object directly using new keyword:


    Syntax to create object:
    var object_name = new Object();
    Note: Object is a keyword.

    Syntax to define the properties:
    object_name.property1 = value1;
    object_name.property2 = value2;
    object_name.property3 = value3;

    Note: we cannot access or display the full object. We can only access or display an object, property
    by property.
    Syntax to access object, property by property:
    object_name.property

    Syntax to remove a property from an object:
    delete object_name.property

    Note: after deleting a property we can no longer access it.
