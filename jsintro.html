    What is JavaScript?


    →JS is a client-size scripting language.
    →JS runs on the browser on the client device.
    →Browser understands JS and runs it.
    →JS is lightweight & no download or installation is needed.
    →JS is designed for network based applications i.e. applications that require a network.
    For ex:
    websites.
    →JS is free and open source.
    →JS works on all devices.
    →JS is used:
    →to interact with the user’s device.
    →to create dynamic web pages.
    →to validate form data.
    →to control the client browser.
