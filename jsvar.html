JS-VARIABLE


    →Syntax to declare and initialize a variable:
    var variable_name = value;
    →Note:
    →we do not explicitly specify the data type while declaring a variable.
    →the value that we assign decides the variable’s data type.
    →for example:
    var name = "Sid"; here, name is a String variable as we have assigned a string value to it.
    var age = 33; here, age is an integer variable as we have assigned an integer value to it.
    →we can change the data type of a variable anytime we want by assigning a new value of
    another data type to the same variable.
    →Syntax to display a variable’s value along with some message:
    document.write(“Message” + var_name + “Message”);
    Note: + is the concatenation operator.
