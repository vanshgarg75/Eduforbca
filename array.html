JS-Arrays



    →Array is a special variable.
    →Here, we can store multiple values in a single variable.
    Array Syntax:
    →Syntax 1 to declare and initialize an array:
    var array_variable = [value1, value2, value3,…]

    →Syntax 2 to declare and initialize an array:
    var array_variable = [
    value1,
    value2,
    value3,
    …
    ]

    →Syntax 3 to declare and initialize an array using the new keyword:
    var array_variable = new Array(value1, value2, value,…)

    →Note: every array element has a unique index (serial number) starting from 0.

    →Syntax for Accessing individual array elements:
    array_variable[index]

    →Syntax to change / update individual array elements: array_variable[index] = new_value

    →Syntax to access the size / length of the array:
    var result_variable = array_variable.length

    →Syntax to perform sorting on an array:
    var new_array_variable = array_variable.sort()
