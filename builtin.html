JS-Array Functions



    toString():
    →converts an array to a string of comma separated values.
    →Syntax:
    string_var = array_var.toString();

    join():
    →this function combines all the array elements into a string.
    →here, we can decide the separator.
    →Syntax:
    string_var = array_var.join(“separator”);
    pop():
    →this function is used to remove the last item from an array.
    →Syntax:
    var removed_item = array_var.pop();

    push():
    →this function adds a new item to the end of the array.
    →after inserting the item, this function gives us the new updated length of the array.
    →Syntax:
    var new_length_var = array_var.push(item);
    shift(): →this function removes the 1st item of the array and shifts all the other items to 1 position less. →Syntax: var removed_item = array_var.shift();

    unshift():
    →this function adds a new item to the beginning of the array.
    →after inserting the item, this function gives us the new updated length of the array.
    →Syntax:
    var new_length_var = array_var.unshift(item);

    delete:
    →using this we can delete any item from the array.
    →Syntax:
    delete array_var[index];
    →This will leave a hole in the array at that position.


    splice() {to add new items in the array}:
    →Syntax:
    array_var.splice(insertion_index , no_of_items_to_be_removed , item1, item2, item3);
    →insertion_index: is the index of the array element where the new items need to be added.
    →no_of_items_to_be_removed: how many items should be removed.

    sort() :
    →Sorts the contents of the array in ascending order.
    →Syntax:
    var new_sorted_array = array.sort();
    →After sorting, a new array is returned.

    reverse():
    →this function reverses the elements of the array.
    →Syntax:
    var new_reversed_array = array.reverse();
    →This returns a new reversed array.

    concat():
    →this function is used to concatenate an array at the end of another array.
    →Join at end.
    →Syntax:
    var result_array = array_1.concat(array_2);
    →returns the new array that has array_1 + array_2 elements.
    →array_2 will be added at the end of array_1.

    indexOf():
    →this function is a search function.
    →This function searches for an item in the array.
    →when item is found, it returns the index of that item.
    →when item is not found, it returns -1.
    →syntax:
    var item_index = array.indexOf(item);

    filter():
    →this function is used to apply a user defined / predefined function on an array’s elements.
    →filter() creates a new array filled with elements that all pass a test that is provided by a function.
    →filter() returns a new array.
    →filter() does not change the original array value.
    →Syntax:
    new_array_variable = array_variable.filter( function_name );
