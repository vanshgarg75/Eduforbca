Pragraph in HTML

A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph.

Example

<p>This is a paragraph.</p>

<p>This is another paragraph.</p>

Text Styles in HTML

  • <b>Text</b> used to bold text
  • <i>Text</i> used to make text italic
  • <u>Text</u> underlines the text
  • <strong>Text</strong> similar to bold but differs from browser to browser
  • <em>Text</em> (emphasis) - similar to italics but differs from browser to browser
  • <sup>Text</sup> superscript (23 = 8)
  • <sub>Text</sub> subscript (H2O)
  • <pre>Text</pre> Text in a pre element is displayed in a fixed-width font, and it preserves both spaces and line breaks
