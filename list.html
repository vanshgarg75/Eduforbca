Lists

Type of list:

  1. Ordered List.
  2. Unordered List.
  3. Definition List.

Unordered List:

    →bulleted lists.
    →tags used: <ul></ul> and <li></li>
    →<ul> unordered list
    →<li> list item
    →<ul> attribute: type=”circle” or “disc” or “square”
    →Syntax:
                <ul>
                <li>Item</li>
                <li>Item</li>
                <li>Item</li>
                </ul>

Ordered Lists:

→numbered lists
→tags used: <ol></ol> and <li></li>
→<ol> ordered list
→<li> list item
→<ol> attribute: type=”1” or “A” or “a” or “I” or “i”
    start=”give the start position”
    reversed this will display the numbering in reverse order
→Syntax:
                    <ol>
                    <li>Item</li>
                    <li>Item</li>
                    <li>Item</li>
                    </ol>

Definition List:

→this list is used to define some entities.
→tags used: <dl></dl> and <dt></dt> and <dd> and </dd>
→<dl> definition list
→<dt> definition term
→<dd> definition data
→Syntax:
        <dl>
        <dt>Term</dt>
        <dd>Definition</dd>
.
