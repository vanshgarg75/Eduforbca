CSS Font Properties:

    CSS style options to apply on text: Style Option Description font-family Arial font-style font-weight font-variant font-size
    :the font style to be set on some text. :Time New Roman :| italic | oblique thickness of the font normal | bold. | small-caps size of the text. in px 
                Example:
            <!DOCTYPE html>
            <html>
            <head>
            <title>CSS Font Properties</title>
            </head>
            <body>
            <h1>CSS Font Properties</h1>

            <h2 class="different-font">This is written in a different font family.</h2>

            <p class="italic-bold">
            This is written in italics & bold.
            </p>

            <h2 class="small-caps">This Is Written In Small Caps</h2>

            <p class="big-text">
            This is big text
            </p>
            </body>
            </html>
