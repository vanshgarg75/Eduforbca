External CSS

  • Here, we do not define the CSS styling inside a web page.
  • It is done on a separate (.css) file.
  • We cannot run this file, we can only import this file in an HTML file.
  • Main benefit is reusability of styles across the website (in multiple web pages of the website).
  • Step 3. Import the .css file inside the <head> tag of HTML file:
    •             <head>
                <link rel="stylesheet" type="text/css" href="cssfilename.css"/>
            </head>
  • After this step, we can normally use the CSS classes & styles.
