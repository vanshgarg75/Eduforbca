JS-Form Interaction


    →Access HTML form’s data in JavaScript.
    Step 1: Design the form + the area where the output will be displayed:
    →must have the name attribute of all the form elements.

    Step 2: define a JavaScript function (in the head section) and access the form data
    inside it.


    →the function must get the form as an argument.
    function function_name(form)
    {
    }

    →access the form’s element’s data inside the function using this syntax:
    var variable_name = form.input_type_name.value;

    Note: the input_type_name is the same name that we give while create the form element:
    <input type=”text” name=”input_type_name” />

    Step 3: call the function on the button click (use onclick event) and pass the entire form:

    →syntax: <input type=”button” onclick=”function_name(this.form)” />
