Internal CSS:

  • Different ways to apply CSS on HTML tags
  • Inline CSS
  • Internal CSS
  • External CSS
  • Here, we can give the style in the head instead of on the tag.
    •             <head>
            <style type="text/css"> 
            .classname 
            { 
                css_style_attribute: value; 
                css_style_attribute: value; 
                css_style_attribute: value; 
            } 
            </style> 
            </head>
  • Note: This creates your own custom style which you can later apply on any tag.
    •             <tag class="classname">
  • Applying CSS on specific tags with specific ids:
    •             <head> 
            <style type="text/css"> 
            #tag_id 
            { 
                css_style_attribute: value; 
                css_style_attribute: value; 
                css_style_attribute: value; 
            } 
            </style> 
            </head>
  • Note: This style gets auto set on any tag with that id:
    •             <tag id="tag_id">
  • Applying CSS on all occurrences of a tag:
    •             <head> 
            <style type="text/css"> 
            < Tag_Name> 
            { 
                css_style_attribute: value; 
                css_style_attribute: value; 
                css_style_attribute: value; 
            } 
            </style> 
            </head>
  • Note: This style gets auto set on any tag in the web page:
    •             <tag>
